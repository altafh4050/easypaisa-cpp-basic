# easypaisa-cpp-basic
C++ console simulation of a mobile wallet system (Easypaisa) with accounts, transfers, and bill payments.

# Easypaisa C++ Simulation
This is my first C++ project simulating a mobile wallet system similar to Easypaisa. 
Users can create accounts, login, deposit/withdraw money, transfer funds, and pay bills like electricity, gas, and education fees. 
It is a console application designed to practice C++ and basic OOP concepts.

## Features
- Account creation and login
- Deposit and withdraw money
- Money transfer
- Bill payments (Electricity, Gas, Education)
